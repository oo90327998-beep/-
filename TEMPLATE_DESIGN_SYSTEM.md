### 3. 配色方案
- **主色**：深紫色 `#7c3aed` / 深蓝色 `#1e3a5f` / 深绿色 `#047857`
- **辅助色**：浅紫色 `#a78bfa` / 浅蓝色 `#60a5fa` / 浅绿色 `#34d399`
- **强调色**：暖金色 `#fbbf24` / 琥珀色 `#f59e0b`
- **文字色**：深灰 `#1f2937` / 中灰 `#6b7280` / 浅灰 `#9ca3af`

### 4. 字体规范
- **中文字体**：Microsoft YaHei, PingFang SC, Noto Sans SC
- **英文字体**：Inter, Roboto, Arial
- **字重**：标题 700-800，正文 500-600
- **字间距**：标题 2-4px，正文 0-1px

## 🔧 实现要点

### 1. 统一的HTML结构

```html
<div class="resume-template [template-id]">
  <!-- 顶部横幅 -->
  <div class="resume-banner">
    <div class="banner-content">
      <div class="banner-left">
        <h1 class="banner-title-cn">个人简历</h1>
        <p class="banner-title-en">Personal Resume</p>
        <p class="banner-slogan">[模板口号]</p>
        <div class="banner-icons">
          <span class="icon-gold">[图标1]</span>
          <span class="icon-gold">[图标2]</span>
        </div>
      </div>
      <div class="banner-right">
        <div class="avatar-circle">
          <span>👤</span>
        </div>
      </div>
    </div>
    <div class="banner-accent"></div>
  </div>
  
  <!-- 内容区域 -->
  <div class="resume-body-modern">
    <section class="resume-section-modern">
      <div class="section-block">
        <span class="block-pattern"></span>
        <h3>[模块标题]</h3>
      </div>
      <div class="section-grid">
        <div class="grid-item">
          <span class="label">[标签]：</span>
          <span class="value">[内容]</span>
        </div>
      </div>
    </section>
  </div>
</div>
```

### 2. 统一的CSS类名

- `.resume-banner` - 顶部横幅
- `.banner-title-cn` - 中文大标题
- `.banner-title-en` - 英文副标题
- `.banner-slogan` - 口号
- `.banner-icons` - 装饰图标容器
- `.icon-gold` - 金色圆形图标
- `.avatar-circle` - 圆形头像
- `.banner-accent` - 装饰线
- `.section-block` - 块状标题
- `.block-pattern` - 斜纹图案
- `.section-grid` - 网格布局
- `.grid-item` - 网格项

### 3. 响应式设计

```css
@media (max-width: 768px) {
  .section-grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (max-width: 480px) {
  .section-grid {
    grid-template-columns: 1fr;
  }
}
```

## 🎯 设计原则

1. **统一性**：所有模板遵循相同的逻辑框架
2. **独特性**：每个模板有独特的配色和元素
3. **专业性**：保持专业、清晰、易读
4. **灵活性**：适应不同行业和职位类型
5. **美观性**：注重视觉呈现和用户体验
